Project Outline: The first stage of my project is to present a literature review and comparison of current state-of-the-art methodologies for distributed graph colouring. The second stage will be to implement one such algorithm, recently designed by Maus. The algorithm is round based, allowing nodes in a graph to have a sequence of colours, moving down the sequence until no conflicts exist with its neighbours. This allows nodes with no common neighboursto be processed in parallel. This algorithm creates a colouring bounded below by \delta + 1 and above by O(\delta^2) and Maus further expands on a on-round algorithm to reduce the colourization. Once implemented, benchmarks will be run to see how the algorithm performs in practice. The algorithm may be implemented specifically for GPU hardware to test performance in that case. The benchmarks will be used to compare the algorithm to a modified sequential version and see if the expected results reported by Maus are achieved as well as to see how the algorithm performs in practice.

Startup Paper(s): Maus, Y. (2021). Distributed Graph Colouring Made Easy. Proceedings of the 33rd ACM Symposium on Parallelism in Algorithms and Architectures. https://doi.org/10.1145/3409964.3461804
